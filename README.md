# VibeImpostor
Es el juego del impostor que se ha viralizado, para jugar online con amigos. Se puede jugar por llamada privada (por ejemplo tu haces tu llamada por whatsapp o discord, herramientas externas) mientras usas la aplicacion para gestionar los ajustes, puntos y reparto de palabras, o tambien para jugar escribiendo las palabras por un chat, por si es requerido estar en silencio o no puedes estar en llamada. Tambien puedes usarlo en persona y cada uno usar su propio telefono movil, las posibilidades son infinitas.
